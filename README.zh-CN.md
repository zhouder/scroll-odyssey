# Scroll Odyssey 🧳

把你每天滚动网页的距离变成一场离线旅行。

## 功能

每滚动一个像素，你就在一条真实路线上前进一步。滚动足够多，你就能"走完"阿巴拉契亚山径、骑行环绕台湾，或者穿越撒哈拉——一次网页滚动一步。

- **10 条内置路线** — 城市马拉松、长城徒步、东京漫步、巴黎漫步、丝绸之路、台湾环岛、圣地亚哥之路、阿巴拉契亚山径、撒哈拉穿越、北京胡同漫游
- **精准距离** — 基于 96 DPI 标准换算（1 米 ≈ 3780 像素），支持 cm / m / km 显示
- **里程碑** — 进度到达关键地点时自动解锁
- **旅行明信片** — 为任意路线生成并下载 PNG 明信片
- **7 天趋势图** — 一眼看清滚动历史
- **连续天数** — 每日连续记录 🔥
- **语言切换** — English / 中文
- **完全离线** — 无服务器、无账号、无埋点

## 安装

### 从源码构建

```bash
git clone https://github.com/yourname/scroll-odyssey
cd scroll-odyssey
npm install
npm run build
```

1. 打开 `chrome://extensions/`（或 `edge://extensions/`）
2. 开启右上角**开发者模式**
3. 点击**加载已解压的扩展程序** → 选择 `dist/` 目录

## 使用

- 滚动任意网页，工具栏图标徽章显示今日公里数
- 点击图标查看今日叙事和路线进度
- 点击 ⚙ 打开仪表盘（总览 / 路线 / 明信片 / 设置）

## 权限说明

| 权限 | 用途 |
|---|---|
| `storage` | 在本地保存滚动数据和设置 |
| `scripting` | 在网页中注入内容脚本以监听滚动 |
| `host_permissions: <all_urls>` | 允许在所有 http/https 页面运行 |

## 隐私承诺

- ✅ 仅保存**距离数字和日期**，不保存网页内容、网址或表单数据
- ✅ 域名统计**默认关闭**，可在设置中手动开启
- ✅ **零网络请求** — 数据永不离开你的浏览器
- ✅ **一键清空** — 设置页 → 清空所有数据
- ✅ 自动跳过 `chrome://`、`edge://` 及扩展页面

所有数据仅存储在本机的 `chrome.storage.local` 中。

## 距离换算说明

> 1 像素 ≈ 0.2646 mm（96 DPI 标准）→ 1 米 ≈ 3780 像素

实际值受屏幕 DPI 和系统缩放影响，此为合理估算值。
